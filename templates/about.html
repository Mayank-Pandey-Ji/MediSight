About This Project

MediSight is an AI-powered health prediction web application designed to help people understand possible diseases based on their symptoms. This tool acts as a quick and early assistant, helping users become more aware of their health.

How It Works

This website uses a Machine Learning model (Support Vector Machine – SVM) that has been trained on a medical dataset containing:

  • Symptoms of various diseases
  • Description and details of each disease
  • Precautions and preventive measures
  • Common medications
  • Recommended diet charts
  • Workout & lifestyle recommendations

Our Goal

The purpose of MediSight is to provide a quick overview of what a person might be experiencing, based on the symptoms they input. This is meant to help users become informed and take the next steps toward better health.

Important Note

This is an educational and informational tool. It is not a substitute for a medical professional. Always consult a certified doctor for diagnosis, medication, and treatment.

Technology Used

  • Python (Flask Framework)
  • Machine Learning (Support Vector Machine Model)
  • Pandas & NumPy for preprocessing
  • TailwindCSS for beautiful UI
  • Frontend templating using Jinja2
